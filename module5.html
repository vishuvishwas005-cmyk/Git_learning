Module 5: Advanced Git Features

Rebasing

Rebasing rewrites commit history by moving commits to a new base:

git checkout feature-branch
git rebase main

Interactive rebase to edit commits:

git rebase -i HEAD~3

Stashing Changes

Temporarily save uncommitted changes:

git stash

Apply stashed changes:

git stash apply

List stashes:

git stash list

Resetting and Reverting

Soft reset (keep changes):

git reset --soft HEAD~1

Hard reset (lose changes):

git reset --hard HEAD~1

Revert a commit (creates new commit):

git revert commit-hash

Cherry Picking

Apply specific commits from another branch:

git cherry-pick commit-hash

Resolving Merge Conflicts

When conflicts occur:

  1. Identify conflicted files
  2. Edit files to resolve conflicts
  3. Stage resolved files
  4. Complete the merge/rebase

Git Bisect

Find the commit that introduced a bug:

git bisect start
git bisect bad
git bisect good commit-hash

Git Hooks

Automate tasks with scripts in .git/hooks/:

  • pre-commit: Run before committing
  • post-commit: Run after committing
  • pre-push: Run before pushing
