Common Git Questions

How do I undo a commit?

Use git reset --soft HEAD~1 to undo the last commit but keep changes staged, or git reset --hard HEAD~1 to completely remove it.

What does 'git pull' do?

git pull fetches changes from the remote repository and merges them into your current branch.

How do I create a new branch?

Use git checkout -b new-branch-name to create and switch to a new branch.
